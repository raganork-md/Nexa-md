# 🧬 NEXA-MD WHATSAPP BOT
Nexa-MD is a powerful, modular, and simple WhatsApp bot built with the Baileys library.
---
## 🚀 Deployment
### 1. Get Session ID
Scan the QR code or get the Pair Code to obtain your `SESSION_ID`. Use the link below:
👉 [**SCAN QR / GET PAIR CODE**](https://only-cathrine-menualive-c76dd2b7.koyeb.app/)
### 2. Deploy to Koyeb
1. Fork this repository.
2. Create a new service on [Koyeb](https://app.koyeb.com/).
3. Connect your GitHub repository.
4. Add the following **Environment Variables**.
---
## ⚙️ Environment Variables

| Key | Description |
| :--- | :--- |
| `SESSION_ID` | Your generated Nexa-MD Session ID |
| `OWNER_NUMBER` | Your WhatsApp number (e.g., 919876543210) |
| `OWNER_NAME` | Your Name |
| `PREFIX` | Bot prefix (e.g., `.`) |
| `MODE` | Bot mode (`public` or `private`) |
| `BOT_NAME` | Name of your bot (Default: Nexa-MD) |

---
## 🛠️ Features
- **Modular Plugins:** Easily add or remove commands.
- **Variable Management:** Change settings using `.setvar`.
- **Public/Private Modes:** Control who can use the bot.
- **Fast Response:** Optimized for low latency.
